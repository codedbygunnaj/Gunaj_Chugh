# 🚀 Gunaj Chugh - Portfolio v2

A modern, responsive developer portfolio built with React, showcasing my work, experience, and projects in a clean and consistent UI.

🔗 Live: 

---

## ✨ Features

- ⚡ Clean and modern dark UI
- 📱 Responsive layout with sidebar navigation
- 🧠 Dynamic typing effect in hero section
- 📂 GitHub-powered projects section
- 🧩 Structured experience timeline
- 📬 Functional contact form (backend integrated)
- 🎯 Consistent design system across all pages

---

## 🛠 Tech Stack

- **Frontend:** React, CSS3, Bootstrap (base)
- **Routing:** React Router
- **Backend (Contact Form):** Node.js + Express (deployed on Render)
- **Deployment:** Vercel
